Bem-vindo ao MiniOS.

Selecione uma opção pré-configurada no menu de inicialização ou pressione a tecla [Tab] para editar manually a linha de comando
de inicialização e personalizar como o MiniOS inicializa. Uma descrição detalhada dos vários parâmetros de inicialização é
fornecida abaixo.

* text: Inicia no modo de texto, sem interface gráfica.
* automount: Ativa a montagem automática de dispositivos de armazenamento.

* toram: Copia todos os dados para a RAM, o que permite ejetar o dispositivo de inicialização.
  - toram=full: Copia tudo para a RAM.
  - toram=trim: Copia para a RAM apenas os módulos filtrados.

* nozram: Desativa a compressão de RAM (zRAM).
* zramcomp=[lzo|lzo-rle|lz4|lz4hc|zstd]: Define o algoritmo de compressão da zRAM.
* zramsize=[MB]: Define o tamanho da zRAM em megabytes.

* from=[caminho/dispositivo/URL]: Especifica a localização dos dados do MiniOS.
  - from=/minios: Usa o diretório /minios.
  - from=/dev/sda1/minios: Usa um dispositivo e um caminho específicos.
  - from=http://server/file.iso: Baixa de um servidor HTTP.
  - from=askdisk: Permite selecionar o disco de forma interativa.

* perch: Ativa as alterações persistentes (requer um dispositivo de armazenamento gravável).
* perchdir=[resume/new/ask/#]: Escolhe uma sessão de persistência.
  - resume: Retoma automaticamente a última session.
  - new: Sempre cria uma nova sessão.
  - ask: Permite selecionar a sessão de forma interativa.
  - #: Usa um número de sessão específico.
* perchmode=[native/dynfilefs/raw]: Define o modo de persistência.
  - native: Sistema de arquivos nativo (padrão, melhor desempenho, apenas POSIX).
  - dynfilefs: Contêiner de arquivo dinâmico (funciona em todos os sistemas de arquivos).
  - raw: Imagem raw de tamanho fixo.
* perchsize=[MB]: Define o tamanho do contêiner de persistência em megabytes.

* load=[filtro]: Carrega apenas os módulos correspondentes (regex ou intervalo).
  Exemplos: load=00-core ou load=core,kernel,firmware ou load=00,01,02
* noload=[filtro]: Exclui os módulos correspondentes.
  Exemplos: noload=05-xfce-apps ou noload=xfce-apps,firefox ou noload=05,06

Para mais informações, visite https://minios.dev/docs
